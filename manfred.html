Manfred Awesomic CV (MAC) is an open source project for Manfred, which allows you to build a JSON-based resume. Created during intensive frontend development course in Adalab in collaboration with Emma Martin, Marina Ramos y Laura Ferrando.
You can create your resume in three different ways:
1. Fill out the form
2. Modify JSON file
3. Create your own JSON and paste it to the editor and the output will appear in CV viewer immediately
When you are done with your resume you have two options: print it/save it as PDF file or you can copy your resume as a html/css code and paste it in your own project or wherever you want ;)
